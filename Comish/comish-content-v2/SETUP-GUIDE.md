# HOW TO SCREENSHOT GRAPHICS

All graphics are HTML files. To export:

1. Open `.html` file in **Chrome**
2. Right-click ‚Üí **Inspect** (or Cmd+Shift+I)
3. Click **device toolbar** icon (or Cmd+Shift+M)
4. Set dimensions:
   - Logo: 400 x 400
   - LinkedIn Company Banner: 1128 x 191
   - Twitter Header: 1500 x 500
   - Post Graphics: 1080 x 1080
   - Carousel Slides: 1080 x 1350
5. Screenshot (Cmd+Shift+4 on Mac)

**Pro tip:** Set device pixel ratio to 2x for higher quality.

# CONTENT VOICE

Since these are brand accounts (not personal), use this voice:

```
- Confident but not corporate
- Helpful, practical
- Slightly irreverent ("spreadsheets suck")
- Use "we" occasionally, but often just state facts
- Minimal emoji (üü¢ as brand accent, occasional üìäüí∞)
```

### Twitter (Most Important)
- Reply to sales influencers' tweets

### Instagram
- Use Stories for behind-the-scenes
- Reply to every comment and DM
- Use hashtags strategically

- Lowest priority - just maintain presence
- Repurpose content from other platforms
- Consider joining sales groups and engaging there
