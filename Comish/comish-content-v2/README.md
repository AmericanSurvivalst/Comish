## üìê HOW TO USE THE GRAPHICS

All graphics are HTML files. To export them:

1. Open the `.html` file in **Chrome** (drag file into browser)
2. Right-click ‚Üí **Inspect** (or Cmd+Shift+I / Ctrl+Shift+I)
3. Click the **device toolbar** icon (or Cmd+Shift+M)
4. Set dimensions to match the graphic (usually 1080x1080 or 1080x1350)
5. **Screenshot** (Cmd+Shift+4 on Mac, or use a browser extension)
6. Post to social media

**Pro tip:** For highest quality, set device pixel ratio to 2x in device toolbar.
